# genai-portfolio

My GenAI & Agentic AI learning projects and portfolio.

## Structure
- week0/  - setup & git practice
- week1/  - embeddings & NLP
- week2/  - vector DBs & semantic search
- week3/  - LLMs, prompting, RAG
- week4/  - agents, deployment, portfolio

## How to run
1. Create virtualenv: `python -m venv venv`
2. Activate: `source venv/bin/activate` (Windows: `venv\Scripts\activate`)
3. Install: `pip install -r requirements.txt`
4. Run examples from the `week1/` folder.
